# Screenshots for README / Gumroad / landing

## Placeholder assets (ship now)

Vector placeholders live in **`public/marketing/`** so:

- **`npm run dev`** / production builds serve them at **`/marketing/*.svg`** (for `landing.html`).
- **GitHub README** can embed them with paths like **`public/marketing/panel-placeholder.svg`**.

| File | Use for |
|------|---------|
| `public/marketing/panel-placeholder.svg` | Three-column live panel |
| `public/marketing/home-placeholder.svg` | Home / curtain screen |
| `public/marketing/env-placeholder.svg` | Missing API key banner (support docs) |

Replace these with **compressed PNG or WebP** when you have real captures (same folder + filenames, or update `README.md` and `public/landing.html`).

## Capture tips

- Blur or crop any **API keys** or private topics.
- **1280×720** or **1920×1080** works well on Gumroad.
- Keep total image size small so the repo zip stays light for buyers.
